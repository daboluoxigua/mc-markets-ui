# @mc-markets/ui 使用示例

## 1. 安装

```bash
npm install @mc-markets/ui
# 或
yarn add @mc-markets/ui
# 或
pnpm add @mc-markets/ui
```

## 2. 基础使用

### 完整导入（推荐）

```typescript
// main.ts
import { createApp } from 'vue'
import App from './App.vue'
import McMarketsUI from '@mc-markets/ui'
import '@mc-markets/ui/style' // 导入样式

const app = createApp(App)

// 使用默认配置
app.use(McMarketsUI)

app.mount('#app')
```

### 按需导入

```typescript
// main.ts
import { createApp } from 'vue'
import App from './App.vue'
import { Dialog, Icon, Tab, TabPane } from '@mc-markets/ui'
import '@mc-markets/ui/style'

const app = createApp(App)

// 手动注册组件
app.component('Dialog', Dialog)
app.component('Icon', Icon)
app.component('Tab', Tab)
app.component('TabPane', TabPane)

app.mount('#app')
```

## 4. 主题定制

### 使用主题管理器

```vue
<template>
  <div>
    <h3>主题定制</h3>
    <div class="theme-controls">
      <el-button @click="toggleDark">切换暗色模式</el-button>
      <el-button @click="setBlueTheme">蓝色主题</el-button>
      <el-button @click="setGreenTheme">绿色主题</el-button>
      <el-button @click="resetTheme">重置主题</el-button>
    </div>
  </div>
</template>

<script setup>
import { themeManager } from '@mc-markets/ui'

const toggleDark = () => {
  themeManager.toggleDark()
}

const setBlueTheme = () => {
  themeManager.setTheme({
    name: 'blue',
    dark: false,
    cssVars: {
      '--el-color-primary': '#1890ff',
      '--el-color-success': '#52c41a',
      '--el-color-warning': '#faad14',
      '--el-color-danger': '#ff4d4f',
      '--el-color-info': '#1890ff'
    }
  })
}

const setGreenTheme = () => {
  themeManager.setTheme({
    name: 'green',
    dark: false,
    cssVars: {
      '--el-color-primary': '#52c41a',
      '--el-color-success': '#73d13d',
      '--el-color-warning': '#faad14',
      '--el-color-danger': '#ff4d4f',
      '--el-color-info': '#1890ff'
    }
  })
}

const resetTheme = () => {
  themeManager.setTheme({
    name: 'default',
    dark: false,
    cssVars: {}
  })
}
</script>

<style scoped>
.theme-controls {
  display: flex;
  gap: 12px;
  flex-wrap: wrap;
}
</style>
```

## 6. 注意事项

1. **样式导入**：确保导入了 Element Plus 的样式文件 `@mc-markets/ui/style`
2. **组件注册**：使用完整导入时，组件会自动注册；使用按需导入时需要手动注册
3. **主题切换**：主题切换是全局的，会影响整个应用
4. **类型支持**：框架提供完整的 TypeScript 类型支持
5. **兼容性**：需要 Vue 3.3+ 和 Element Plus 2.8+

这个 UI 框架提供了丰富的组件和功能，让您可以快速构建现代化的 Vue 3 应用。
