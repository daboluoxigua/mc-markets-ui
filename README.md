## 🚀 快速开始

### 安装
```bash

# 生成凭证
git config --global credential.https://gitlab.mc-markets.com.username 用户名

# 安装依赖
npm i git+https://gitlab.mc-markets.com/lty/mcmarkets-ui.git#master

npm i git+https://gitlab.mc-markets.com/lty/mcmarkets-ui.git#v1.0.0
```

### 开发模式

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

### 构建发布

```bash
npm run publish:lib
```
